Vezérlési logika

A termosztát folyamatosan értékeli az aktuális hőmérsékletet, a célértéket és az ütemezést, majd ennek alapján dönt a fűtés be- vagy kikapcsolásáról.

Automatikus mód

Ütemezés alapú vezérlés (hétköznap / hétvége vagy haladó ütemterv), tanuló algoritmussal kiegészítve.

Kézi mód

Fix célhőmérséklet, a logika csak a biztonsági korlátokat érvényesíti.

Előfűtés (Pre‑Heat)

A rendszer a korábbi felfűtési tapasztalatok alapján előbb indítja a kazánt, hogy a kívánt időpontra elérje a célhőmérsékletet.

Ez az oldal GitHub Pages-en futó, statikus dokumentáció. Az éles vezérlőfelület és logika az ESP32 eszközön működik.
