# ESP Thermostat Pro – How It Works

This repository contains a **multi-page GitHub Pages documentation site** in `/docs` built with the **Just the Docs** theme.

- GitHub → **Settings → Pages** → **Deploy from a branch** → `main` → `/docs`
- The Pages site is **documentation only** (no live ESP calls, no cloud uploads).

Language entry points:
- HU: `docs/hu/overview.md`
- EN: `docs/en/overview.md`
